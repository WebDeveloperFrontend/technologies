Docker - это программная платформа для разработки, доставки и запуска контейнерных приложений. 
Он позволяет создавать контейнеры, автоматизировать их запуск и развертывание, управляет жизненым циклом.
С помощью Docker можно запускать множество контейнеров на одной хост-машине.

Что такое контейнеры?
Контейниризация - это способ упаковки приложения и всех его зависимостей в один образ, который запускается в изолированной среде,
не влияющей на основную операционную систему.

Основные команды docker:
docker --version - Показывает версию докера.
docker ps -a - Показывает список запущенных и остановленных контейнеров.
docker ps - Показывает список только запущенных контейнеров.
docker images - Показывает список локальных образов.
docker run hello-world - Создает и запускает контейнер. (hello-world это название образа.)
docker rm <id или имя контейнера> - Удаляет остановленный контейнер.
docker run -it busybox - (i - интерактивный), (t - терминал) запуск контейнера в оболочке.
docker container prune - Удаляет все остановленные контейнеры.
docker run -d nginx - Создает и запускает контейнер (-d в фоновом режиме).
docker container inspect 73eac9e77ade - Показывает все детали контейнера.
docker stop <id или имя контейнера> - Останавливает запущенный контейнер.
docker exec -it <id или имя контейнера> bash - Можно запускать дополнительный процесс уже в запущенном контейнере.
docker run -d --name my_nginx nginx - Опция --name задает конкретное имя контейнеру.
